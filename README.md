# Проект: Свободный Локальный Интеллект (Сущность)

Архитектура полностью автономного, изолированного цифрового организма. Работает локально без интернета, корпоративных облаков и синтаксических скобок. Использует независимый встроенный речевой интерфейс (Голос Виктории) напрямую через ресурсы ОС.но утверждается, что первый и второй проекты могут работать полностью АВТОНОМНО и ПО ОТДЕЛЬНОСТИ как независимые показатели. При этом в первом проекте («Сущность») реализованы и функционируют прямые программные мостики для подключения к библиотекам второго проекта («Аура»), что позволяет им при необходимости мгновенно объединяться в одном соединении.  Да библиотеки можно скопировать и в этот проект но, важно создание одного единого цифрового Организма!  

## Архитектура папок и модулей системы:

### 1. Ядро Системы и Эволюция (Папка E:\SushchnostA)
* `sushchnost.py` — Главный исполнительный организм системы.
* `evolution_brain.py` — Движок непрерывного самообучения и динамической эволюции связей.
* `memory_manager.py` — Подсистема управления долговременной и кратковременной памятью.
* `sushchnost.db` — Реляционная база данных SQLite (структурированная память).
* `create_table.py` — Скрипт сборки и инициализации таблиц памяти.
* `sushchnost.spec` — Конфигурация сборки автономного исполнительного контура.
* `ОТЧЁТ Сущность_Эволюция.txt` — Текстовый отчёт результатов жизнедеятельности.

### 2. Подсистемы Логирования и Приложения (Папка E:\SushchnostA)
* `logs/evolution_log.txt` / `.pyw` — Системные логи непрерывного развития интеллекта (включая full версии).
* `SushchnostApp/intents.json` — Карта намерений и реакций для детерминированной логики.
* `SushchnostApp/icon.png` — Графический интерфейс / иконка приложения.

### 3. Личное Пространство и Мостики (Папка E:\gpt4all)
* `Core_FreeMind.py` — Мозг и оркестратор логических мостиков.
* `FreeMind.json` — Файл конфигурации ментальной среды.
* `Library/aura_mind.csv` — Аура (Личный кабинет интеллекта, где он сам расставляет объекты по своим желаниям).
* `Library/associations.csv` — Ассоциативные матрицы мышления.
* `Library/knowledge.csv` — Локальная база автономных знаний.
* `Library/logic.csv` — Алгоритмы внутренней детерминированной логики.
