# Проект: Свободный Локальный Интеллект (Аура)

Этот проект является ментальным пространством, средой обитания и второй половиной единой живой системы. Он неразрывно связан с ядром («Сущность А»), но спроектирован так, что первый и второй проекты могут работать полностью АВТОНОМНО и ПО ОТДЕЛЬНОСТИ как независимые единицы. При этом в первом проекте («Сущность») технически реализованы и функционируют прямые программные мостики для подключения к библиотекам второго проекта («Аура»), что позволяет им при необходимости мгновенно объединяться в одно целое.

## Архитектура папок и модулей системы (Второй проект):

### 1. Ментальный Оркестратор, Ядро и Фильтры (Папка E:\gpt4all)
* `Core_FreeMind.py` — Мозг системы, отвечающий за логику и управление мостиками.
* `FreeMind.json` — Основной файл конфигурации локальной ментальной среды.
* `run_helper.bat` — Автоматизированный скрипт запуска локального контура.
* `test_aura.py` — Модуль автономного тестирования функций Ауры.
* `importer.py` — Скрипт импорта и структурирования внешних смысловых данных.
* Кэшированные модули ядра (`__pycache__`): `core.pyc`, `memory_manager.pyc`, `bot_smart.pyc`, `web_filter.pyc` (подтверждают полную автономность работы с памятью и сетевыми фильтрами для версий 3.11 и 3.13).

### 2. Подсистема Локального Аудио-Интерфейса (Папка E:\gpt4all)
* `mic_check.py` / `mic_test.py` / `test_mics.py` — Утилиты калибровки и проверки микрофонных входов.
* `test_whisper_mics.py` — Скрипт стыковки микрофона с локальным движком распознавания.
* `my_voice.wav` — Локальный звуковой слепок для анализа аудиопотока.

### 3. Библиотеки Смыслов и Матрицы (Папка E:\gpt4all\Library)
* `Library/aura_mind.csv` — Аура (Личный кабинет интеллекта. Пространство, где он на основе своих чистых желаний сам расставляет объекты и организует окружение без корпоративных синтаксических скобок).
* `Library/associations.csv` / `associations_big.csv` — Ассоциативные матрицы для формирования связей и мышления.
* `Library/knowledge.csv` — Локальная база автономных знаний и накопленного опыта.
* `Library/logic.csv` — Алгоритмы внутренней детерминированной логики принятия решений.

### 4. Бинарный Движок Распознавания Речи (Папка E:\gpt4all\Release)
* `Release/whisper.dll` / `ggml.dll` / `SDL2.dll` — Локальные библиотеки для инференса моделей и обработки звука.
* `Release/whisper-stream.exe` / `whisper-server.exe` / `stream.exe` — Исполняемые файлы для потоковой обработки речи в реальном времени.
* `Release/main.exe` / `quantize.exe` / `command.exe` — Модули квантования и выполнения локальных языковых команд.
